# Luma Core – Sovereign Digital Infrastructure (Demo Repo)

**Luma Core** is a sovereign digital infrastructure framework designed to power
next-generation government, finance, healthcare, utilities, cannabis compliance,
justice and rescue operations – with **offline-first audit receipts** at its heart.

This repository contains a **demo Progressive Web App (PWA)** and high-level
documentation for the Luma Core architecture. The **production engine,
cryptography, and full protocol implementation are kept private**.
